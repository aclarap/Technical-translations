# *Translation Notes*
This document explains the main decisions made during the translation of the API text from English to Brazilian Portuguese.

# Translation Approach
The translation focused on:
* Clarity and readability in Brazilian Portuguese
* Maintaining the original meaning of the text
* Adapting sentence structures to sound natural, not literal

# Terminology Choices
Some technical terms were kept in English due to common usage in the IT field:
* API → kept as "API"
* JavaScript / Python → kept unchanged
* GPU → kept unchanged

Other terms were translated to maintain understanding:
* "Request" → "Requisição"
* "Response" → "Resposta"

## Challenges
Some parts of the text required careful adaptation:
* The electricity example needed to sound natural in Portuguese without becoming too literal
* Balancing technical accuracy with simple explanations
* Avoiding unnatural sentence structures from direct translation

## 🔍 Style Decisions

* Preferred clear and simple sentences
* Avoided overly literal translations
* Maintained a didactic tone, as the text is educational
