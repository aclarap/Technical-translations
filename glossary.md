# Glossário Técnico – Web APIs
| Inglês | Português | Observação |
|--------|----------|-----------|
| API | API | Mantido em inglês (padrão da área) |
| Application Programming Interface | Interface de Programação de Aplicações |
Forma completa da sigla |
| higher-level language | linguagem de alto nível | Termo técnico consagrado |
| low-level code | código de baixo nível | Tradução padrão |
| abstract (verb) | abstrair a complexidade | Evita tradução literal
(“abstrato”) |
| syntax | sintaxe | Tradução direta |
| developer | desenvolvedor | Termo padrão |
| functionality | funcionalidade | Tradução direta |
| power supply | fonte de alimentação | Contexto técnico |
| appliance | aparelho | Pode variar dependendo do contexto |
| plug socket | tomada | Adaptação natural |
| wire (verb) | conectar | Evita tradução literal (“fazer fiação”) |
| GPU | GPU | Mantido em inglês (sigla técnica) |
| graphics functions | funções gráficas | Tradução direta |
| 3D graphics | gráficos 3D | Ordem adaptada ao português |
| programming language | linguagem de programação | Tradução padrão |

# Notas de Tradução
- O termo “API” foi mantido em inglês por ser amplamente utilizado na área de
tecnologia.
- “abstract” foi traduzido como “abstrair a complexidade”, evitando tradução
literal.
- “plug socket” foi adaptado para “tomada”, priorizando naturalidade.
- “wire” foi traduzido como “conectar”, considerando o contexto técnico.
